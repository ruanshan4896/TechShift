This commit introduces four major improvements to enhance user engagement,
performance, and social media visibility:

1. Async View Counter & Popular Posts
   - Created client-side API route for non-blocking view counting
   - Added ViewCounter component that tracks views in background
   - Popular posts sidebar automatically surfaces high-engagement content

2. Related Posts Component
   - Smart tag-based article recommendations
   - Displays 4 related articles at bottom of each post
   - Improves internal linking structure for SEO
   - Increases time-on-site and reduces bounce rate

3. Pagination System
   - Implemented pagination for homepage, category, and tag pages
   - 12 articles per page for optimal performance
   - Smart page number display with ellipsis
   - URL-based state management (?page=2)
   - Significantly improves page load times

4. Enhanced Social Media Meta Tags
   - Comprehensive Open Graph tags for Facebook/LinkedIn
   - Twitter Card tags for rich previews
   - Proper image dimensions (1200x630)
