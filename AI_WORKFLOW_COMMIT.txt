feat: Implement AI-powered content workflow with manual review

This commit introduces a comprehensive admin-triggered, AI-powered content
processing system that transforms RSS feeds into high-quality, SEO-optimized
articles with manual review capabilities.

## Phase 1: Database & UI Enhancements

Database Schema:
- Added 'status' field to articles table (DRAFT/PUBLISHED)
- Created migration for backward compatibility
- Added 10+ new database functions for draft management

Admin Dashboard:
- Two-tab interface (Drafts/Published)
- Real-time count badges
- One-click publishing from drafts
- Status-based filtering
- Enhanced article management

Editor Enhancements:
- Tag management component with multi-select
- Create new tags on-the-fly
- Pre-populated with AI-suggested tags
- Status toggle (Draft/Published)

## Phase 2: On-Demand RSS Processing

RSS Manager:
- "Fetch & Process" button for each RSS source
- Processes 10 latest articles per source
- Duplicate detection and skipping
- Detailed progress and results display

API Endpoint:
- POST /api/process-rss/[sourceId]
- Fetches RSS feed
- Checks for duplicates
- Processes through AI pipeline
- Returns detailed results

## Phase 3: Advanced AI Processing

API Key Rotation System:
- GeminiKeyManager class for key rotation
- Supports up to 20 API keys
- Round-robin distribution
- Prevents rate limiting
- Configurable via GEMINI_API_KEYS env variable

Three-Step AI Pipeline:

Step A - AI Analysis & Extraction:
- Analyzes original article content
- Extracts main keyword
- Generates SEO-friendly slug
- Finds cover image URL
- Suggests 5 relevant tags
- Returns structured JSON

Step B - AI Content Rewriting:
- Rewrites article in Vietnamese (600-900 words)
- Focuses on user intent and SEO
- Structured with H2/H3 headings
- Incorporates LSI keywords naturally
- Generates meta description (150-155 chars)
- Completely unique content

Step C - Internal Linking:
- Queries database for related published articles
- Finds 2-3 articles sharing tags
- Inserts contextual markdown links
- Natural placement in content
- Improves SEO and user engagement

Main Processing Function:
- processArticleWithAI() orchestrates all steps
- Error handling at each stage
- Fallback mechanisms
- Detailed logging

## Workflow

1. Admin clicks "Fetch & Process" on RSS source
2. System fetches 10 latest articles
3. AI processes each article:
   - Analyzes and extracts metadata
   - Rewrites content with SEO optimization
   - Inserts internal links to related articles
4. Saves all as drafts with suggested tags
5. Admin reviews drafts in dashboard
6. Admin edits content/tags as needed
7. Admin publishes when satisfied
